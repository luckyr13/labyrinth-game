# Labyrinth
 A Maze game made in Typescript with Phaser 3.

## Demo
You can play a demo here: https://labyrinth-bgd.pages.dev/

## Algorithms
 - Depth First Search
 - Greedy Best-First Search
 - Breadth First Search

## How to install locally
Clone the repo and then run:

`npm install`

`npm run dev`
